# DB MVP (Demo Version)

## 📌 Описание проекта

Данный репозиторий содержит **демонстрационную версию** учебного проекта
по разработке MVP системы управления базами данных (СУБД).

Проект реализует упрощённую модель работы с базой данных через
консольный интерфейс и библиотеку на C++.

------------------------------------------------------------------------

## 🎯 Цель проекта

Реализация минимально жизнеспособной СУБД с поддержкой базовых
SQL-команд:

-   CREATE DATABASE
-   USE
-   CREATE TABLE
-   INSERT
-   SELECT
-   UPDATE
-   DELETE

------------------------------------------------------------------------

## ⚠️ ВАЖНО

Данный репозиторий содержит **ограниченную (демонстрационную) версию
проекта**.

🔒 Из него намеренно исключены: - полная реализация SQL-парсера; -
логика выполнения запросов; - механизм хранения данных; - ключевые
алгоритмы обработки WHERE и операций над таблицами.

👉 Полная версия проекта находится в приватном репозитории автора.

------------------------------------------------------------------------

## 🏗️ Архитектура

Проект построен по принципу разделения:

-   Интерфейс (API)
-   Парсер SQL (скрыт)
-   Исполнитель запросов (скрыт)
-   Хранилище данных (скрыто)

Паттерн: - Facade --- класс Database
